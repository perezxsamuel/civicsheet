# Civic Sheet

This repository contains the `civicsheet-ui` application, a Vite + React project for building the Civic Sheet user interface.

## Project structure

- `civicsheet-ui/` — main frontend application.
- `package.json` — root npm metadata.
- `package-lock.json` — locked dependency tree.
- `.gitignore` — repository ignore rules.
- `LICENSE` — project license.

## Getting started

From the project root:

```powershell
cd e:\Repo\civic-sheet
npm install
cd civicsheet-ui
npm install
npm run dev
```

Then open the Vite URL shown in the console.
